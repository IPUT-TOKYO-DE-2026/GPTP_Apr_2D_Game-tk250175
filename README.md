# 図形弾

## ゲームの内容
特性の違う３つの図形弾を飛ばして長く生き残ろう！

<img width="240" height="140" alt="image" src="https://github.com/user-attachments/assets/39bcfa89-9609-42f4-b569-e6fabd283734" />

## 操作説明
WASDで前後左右に移動
U 中範囲を持続的に爆発で攻撃(クールタイム2秒)
I 直線状に高速の貫通弾を発射(クールタイム1秒)
O 一般的な弾で連射可能(クールタイムなし)
